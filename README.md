# E-Commerce Platform

A robust, full-stack E-Commerce application with Spring Boot Backend and React Frontend.

## Project Structure

- `e-com/`: Backend Application (Spring Boot)
- `ecom-frontend/`: Frontend Application (React + Vite)

## Quick Start

### Backend
1. `cd e-com`
2. `mvn spring-boot:run`

### Frontend
1. `cd ecom-frontend`
2. `npm install`
3. `npm run dev`

## Features
- **Authentication**: JWT, Role-based (Admin, User).
- **Payments**: Stripe, Wallet, COD.
- **Admin Panel**: Manage products, categories, orders.
